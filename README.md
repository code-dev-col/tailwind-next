6. **Seguridad por defecto**: Sanitizaci√≥n autom√°tica
7. **Estado integrado**: Zustand como primera opci√≥n

### Prioridad de estilos

1. **Estilos base** - Clases fundamentales del componente

3. **Custom** - `$custom` puede sobrescribir todo
4. **ClassName** - Clases adicionales est√°ticas
